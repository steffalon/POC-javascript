# Overview

JsStore is Sql Like IndexedDb Wrapper.It provides simple api to store, retrieve, delete, remove, and for other advanced Database functionalities.

# Features 

* Executes in Web Worker
* Join Support
* DataType Support
* Complex queries support
* api sync support
* TypeScript support
